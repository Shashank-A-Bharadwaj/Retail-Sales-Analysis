# Retail Sales Analysis

### Objective
Analyze sales data to identify top products, regions, and sales trends.

### Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

### Key Insights
- Top-selling categories identified
- High-performing regions analyzed
- Monthly sales trends observed

### Business Recommendations
- Focus marketing and inventory on top-performing categories  
- Strengthen presence in high-revenue regions  
- Introduce promotions during low-performing months  
- Implement region-specific strategies based on category demand
